# हिंदी भाषा — Gyan Academy

कक्षा 6 से 9 तक हिंदी भाषा, व्याकरण और पद्धतिशास्त्र — Theory, Test और PDF सामग्री।

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## 📁 Project Structure

```
├── public/pdfs/       # PDF textbooks (STD6-8, Grammar, Padhtishastra)
├── src/
│   ├── app/           # Next.js pages & routes
│   ├── components/    # Reusable UI components
│   ├── data/          # Chapter theory & test data (TypeScript)
│   ├── lib/           # Utilities, config, hooks
│   └── types/         # TypeScript type definitions
```
